# Cogent-Learn

Available in [www.cogentcore.org](https://www.cogentcore.org/core)
and also in [Github page](https://github.com/cogentcore/core)

The folder of progression is

1. hello-cogent
2. app-base
3. widgets
4. events
5. styling
6. update
7. value-bind
8. plans
9. async
